# 快速开始指南

## 一键启动

### Windows

双击运行 `start.bat`，或在命令行中：

```cmd
cd web
python run.py
```

### Linux/Mac

```bash
cd web
chmod +x start.sh
./start.sh
```

或直接运行：

```bash
cd web
python3 run.py
```

## 访问地址

启动后，在浏览器中打开：

**http://127.0.0.1:5000**

## 功能预览

1. **主页**: 查看所有分析运行的列表
2. **详情页**: 点击任意运行记录查看详细分析报告，包括：
   - 关键指标统计
   - 可视化图表（胜率、收益分布等）
   - 绩效报告表格
   - 交易明细列表
