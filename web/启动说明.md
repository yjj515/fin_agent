# 启动Web服务器

## 方法1: 使用启动脚本（推荐）

### Windows:
双击运行 `start.bat` 文件，或在命令行中：
```cmd
cd web
start.bat
```

### Linux/Mac:
```bash
cd web
chmod +x start.sh
./start.sh
```

## 方法2: 直接运行Python

```bash
cd web
python run.py
```

## 访问地址

服务器启动后，在浏览器中访问：

**http://127.0.0.1:5000**

## 新功能说明

### 用户搜索功能
1. 在页面顶部的搜索框中输入雪球网用户名
2. 点击"搜索"按钮或按回车键
3. 如果找到该用户的运行记录，会显示所有相关记录
4. 如果没有找到，会显示错误提示信息
5. 可以点击"显示所有记录"查看全部运行记录

### 示例用户名
- 慢跑者397
- 基本面转折
- 省心省力

## 停止服务器

按 `Ctrl+C` 停止服务器
